- Abrir un powershell, abrir la ruta
- cd "C:\Users\<USUARIO>\Downloads\HW-SD-Decorator-main\HW-SD-Decorator-main\EXECUTABLE"

- Entrar desde el File Explorer a EXECUTABLE
- Dar doble clic a el ejecutable Frontend
